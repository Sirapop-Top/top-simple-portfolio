# Sirapop Top Portfolio

A modern, responsive personal portfolio website for an Auditor and Accountant, showcasing professional experience and project achievements.

## Features
- Detailed professional experience timeline.
- Clean, accessible design.
- Responsive layout with dark mode toggle.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Deployment
This project is hosted on GitHub Pages, deployed from the `/docs` directory.
