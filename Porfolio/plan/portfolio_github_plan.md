# Project Plan: Professional Auditor Portfolio

## Objective
Create a modern, dark-themed personal portfolio website for an auditor/accountant, hosted on GitHub Pages.

## Specs & Design
- **Theme:** Modern Dark Theme.
- **Typography:** JetBrains Mono font throughout.
- **Key Sections:**
    - **Hero:** Name ("Sirapop Top"), Professional Title ("Auditor & Accountant").
    - **Projects:** 3 highlighted professional/academic projects.
    - **Contact:** Email, LinkedIn, and/or professional contact form.

## Phased Implementation Plan

### Phase 1: Structure & Setup
- [ ] Initialize repository locally.
- [ ] Create `index.html` with semantic structure.
- [ ] Set up `style.css` (CSS Reset, Dark Theme variables).
- [ ] Import JetBrains Mono via Google Fonts.

### Phase 2: Component Development
- [ ] **Hero Section:** Centered layout, typography styling.
- [ ] **Projects Section:** Grid layout with 3 project cards (Title, Description, Skills/Tech).
- [ ] **Contact Section:** Clean list/links to professional profiles.

### Phase 3: Deployment
- [ ] Configure GitHub repository.
- [ ] Enable GitHub Pages (Direct Branch Deployment).
- [ ] Perform final verification of site accessibility.
